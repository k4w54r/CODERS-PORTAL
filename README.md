# CODERS PORTAL
A Social Networking Portal For Coders

## Features
-   Publicly available:
    -   List of all registered coders
    -   Profile of each coder
-   Privately available:
    -   Personal dashboard
        -  Eexperiences section
        -  Education section
    -   Edit Profile
        -   Add Experience
        -   Delete Experience 
        -   Add Educational Institution
        -   Delete Educational Institution
    -   Posts
        -   Create Posts
        -   Delete Posts
        -   Like Posts
        -   Unlike Posts
        -   Comment On Posts
        -   Delete Comments


## Built With

* MongoDB
* Express
* React
* Node
